# Repo Agent Rules

- Varsayilan calisma dali `semester-project` olsun. Kullanici acikca istemedikce `master` uzerinde degisiklik yapma.
- Oturuma baslarken once `docs/codex/PROJECT_CONTEXT.md` ve `docs/codex/SESSION_RESET.md` dosyalarini oku.
- Yeni dokumanlari `docs/` altinda olustur. Visual Studio'da gorunmesi isteniyorsa `AcademicPortfolio.sln` icindeki `docs` solution folder'ina ekle.
- Kod degisikliginden sonra mumkunse `dotnet build AcademicPortfolio.sln` ile dogrulama yap.
- Gercek sir, parola veya baglanti bilgisini repoya commit etme.
- Uzun suren islerde karar, handoff ve inceleme notlarini `docs/` altinda guncel tut.
