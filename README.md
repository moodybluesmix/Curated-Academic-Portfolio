# Curated-Academic-Portfolio

This project provides the backend infrastructure for the Curated Academic Portfolio (CAP) system, designed to help researchers manage publications, citations, projects, and indexing data in a cleaner, more structured, and verifiable way.
The goal is to offer a curated, researcher-centric academic profile that goes beyond existing platforms.


# Technologies Used
.NET 8

ASP.NET Core Web API

Entity Framework Core (PostgreSQL - Npgsql)

Layered Architecture: Domain / Application / Infrastructure / API
