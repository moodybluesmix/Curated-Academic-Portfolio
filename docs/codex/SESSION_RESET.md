# Session Reset

## Fast Reset Prompt

Asagidaki metni yeni bir oturumda dogrudan kullan:

```text
Bu repoda devam ediyoruz. Once AGENTS.md ve docs/codex/PROJECT_CONTEXT.md dosyalarini oku. Sonra git status --short --branch ile mevcut durumu kontrol et. Kisa bir durum ozeti ver ve semester-project dali uzerinde calismaya devam et.
```

## Reset Checklist

1. Dogru dalda ol: `semester-project`
2. Repo kurallarini oku: `AGENTS.md`
3. Kalici baglami oku: `docs/codex/PROJECT_CONTEXT.md`
4. Git durumunu kontrol et
5. Gerekli gorev dokumanini veya inceleme notunu ac
6. Is bitince ilgili dokumani guncelle

## When To Update Docs

- Mimari karar verildiginde
- Yeni kalici workflow benimsendiginde
- Uzun sureli veya tekrarli bir is akisi netlestiginde
- Bir sonraki oturumda hatirlanmasi gereken baglam olustugunda
